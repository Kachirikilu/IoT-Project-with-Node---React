Backend
1. Install Node JS
    node -v
    npm -v
2. npm init -y
3. npm i express mysql2 sequelize cors multer
4. npm install express-fileupload
5. npm install multer
6. Tambahkan "type": "module" di package.json
7. Buat file index.js
8. Install Nodemon
    npm i -g nodemon
    nodemon -v
9. Install Socket.io
    npm install socket.io
10. Running Serve
    nodemon index


Frontend
1. Create React with Vite
    npm create vite@latest frontend
        npm install
        npm i react-router-dom axios
2. Install tailwind
    npm install tailwindcss @tailwindcss/vite
    https://tailwindcss.com/docs/installation/using-vite
3. Install Socket.io
    npm install socket.io-client
4. Install React Router Dom
    npm install react-router-dom
4. Jalankan server
    npm run dev
